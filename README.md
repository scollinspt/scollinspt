### Sean Collins  

Professor & Director, School of Health at Plymouth State University  
Founding Director, Doctor of Physical Therapy Program  

Biomedical engineer, physical therapist, educator, and researcher exploring the intersection of physiology, computation, and philosophy. My work focuses on mechanistic modeling, complex systems, motor control, and scientific realism. Currently building Project RESPOND, an example project based on the respiratory system as part of a longer-term investigation into how causal structures generate observable phenomena and how those structures can be represented mathematically and computationally.
